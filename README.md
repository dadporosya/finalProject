# Бот для мультиплеерных игр

Бот позволяет нескольким пользователям играть в простенькие, пошаговые онлайн игры. Например: игра в города.

## План работы
- Придумать структуру базы данных, которая будет хранить информацию о существующих игровых сессиях.
- Реализовать менеджера базы данных.
- Реализовать методы позволяющие игрокам создавать сессии и наладить обмен информацией между пользователями и сервером.
- Сделать игровую механику.
- Протестировать проект.

- Игра в города
